# Support

## Getting Help

Use the following channels based on your need:

- `GitHub Issues`: bug reports, feature requests, documentation improvements
- `Discussions` (if enabled): ideas, architecture questions, roadmap suggestions
- `SECURITY.md`: private vulnerability reports
- Email: chittsaathi.v1.1@gmail.com

## Before Opening an Issue

1. Check open and closed issues for duplicates.
2. Include reproduction steps and expected vs actual behavior.
3. Share logs or screenshots when relevant.
4. Mention OS, browser, and runtime versions.

## Response Expectations

This is currently a community-maintained project and response times may vary.

We will prioritize:

1. Security and privacy issues
2. Critical defects affecting student workflows
3. Reproducible bugs with clear details
